## ✨ Key Features
- **Brutalist Design System**: A unique, high-contrast aesthetic utilizing raw structural lines, neon highlights, and custom CSS grid patterns.
- **Fully Responsive**: Carefully crafted for both mobile and desktop experiences using Tailwind CSS container queries and responsive modifiers.
- **Modern Tech Stack**: Built entirely with HTML5 and Tailwind CSS, emphasizing raw performance and simplicity.
- **Typography & Iconography**: Uses `Space Grotesk`, `Inter`, and `JetBrains Mono` combined with Google Material Symbols for a striking visual hierarchy.

## 🛠️ Local Development

If you'd like to run this project locally to test or modify the code:

1. **Clone the repository**
   ```bash
   git clone https://github.com/CapAmin22/AI-Product-Manager-.git
   cd AI-Product-Manager-
   ```

2. **Install dependencies** (Optional, for Tailwind CLI / Tools)
   ```bash
   npm install
   ```

3. **Run a local server**
   You can use any live server to view the `index.html` file. For example, using `npx`:
   ```bash
   npx serve .
   ```
   Or if you use VS Code, simply right-click `index.html` and select **"Open with Live Server"**.
